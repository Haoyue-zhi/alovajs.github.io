## react 编码风格的 preact 项目

- preact 的[官方配置教程](https://preactjs.com/guide/getting-started/#aliasing-react-to-preact)。

- 在 vite 中，官方还提供了 **All in one** 的[vite 插件](https://www.npmjs.com/package/@preact/preset-vite)
